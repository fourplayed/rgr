Design: Overflow — Large Depots (20+ assets)

The ring works great up to ~16 assets. Beyond that, dots start overlapping. How should it scale?

MEL 28

A: Concentric Rings

Add a second (and third) ring as count grows. Inner ring = first 8, outer = next 16. Dots shrink slightly on outer rings. Scales to ~40 before needing a fallback.
