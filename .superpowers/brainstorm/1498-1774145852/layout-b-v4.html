Depot Pills Behavior

  • "All Depots" active by default — shows every asset on the map
  • Click a depot pill to filter map to that depot's assets only (deselects "All")
  • Click active depot again to deselect — reverts to "All"
  • Each depot gets a unique colored dot for quick recognition
  • Collapsed sidebar uses abbreviated depot names (BNE, SYD, MEL, PER) to save space
  • Pills populated dynamically from depot data
